# Project Examples (Any Domain)

Use these examples as memory anchors. Pick the closest complexity tier.

## Simple
**Example**: CRUD backoffice.
- Routes: list/create/edit
- Data: Doctrine entities + forms
- Goal: clean controllers, simple validation

## Medium
**Example**: API Platform app with DTOs.
- Routes: resources + filters
- Data: DTOs + processors/providers
- Goal: versioned API, test coverage

## Complex
**Example**: Modular platform with async workflows.
- Routes: API + admin + public
- Data: Messenger, CQRS, multiple bounded contexts
- Goal: scalability, clear domain boundaries
